# Our Handbook

This handbook is based on [Clef's employee handbook](https://github.com/clef/handbook), but heavily amended for use at Icelabz. You are free to fork and use as long as attribution is given to [Clef](https://github.com/clef/handbook) and [icelabz](https://icelabz.co.uk). 

As we are based in the UK the handbook will be heavily amended to suit the UK legislation. Once it has been published to a final format it will be approved by a HR consultant in the UK.
